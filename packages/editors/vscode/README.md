# Visual Studio Code Configuration

XDG-compliant VSCode settings.

## Files

- `.config/Code/User/settings.json` - VSCode user settings

## XDG Compliance

VSCode naturally follows XDG specification on Linux/macOS by storing settings in `~/.config/Code/User/`.

## Settings

- Sidebar on the right
- Activity bar hidden for cleaner interface
- Minimap disabled
- Breadcrumbs disabled
- Block cursor style
- Gruvbox Dark (Medium) theme
- Auto-trim whitespace on save
